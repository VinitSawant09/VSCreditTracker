# VSCreditTracker

Technologies used:

1) Java 1.8

2) Spring MVC 5.1

3) Hibernate ORM 5.4.21

4) HSQL DB 2.4.0 (In memory database)

5) JUnit 5

6) JavaScript (Vanilla) , AJAX

7) HTML, CSS.

Use username = admin, password = admin for admin login. (Default in database)
